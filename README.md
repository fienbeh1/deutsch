# Deutsch Lernen
German Course Materials

## Quick Start

```bash
git clone https://github.com/fienbeh1/deutsch.git
cd deutsch

# Install LFS for large files
git lfs install
git lfs track "*.pdf" "*.mp3" "*.m4a" "*.iso" "*.zip"
git lfs pull

# Run
npm install
node backend/server2.js
```

Open http://localhost:3456
